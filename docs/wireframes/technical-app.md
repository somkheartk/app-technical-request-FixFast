## Notes

### Technician App Features

1. **Registration & Verification**
   - Document verification
   - Background checks
   - Skills assessment

2. **Job Management**
   - Real-time job notifications
   - Accept/reject jobs
   - Track multiple jobs
   - Update job status

3. **Navigation**
   - GPS navigation
   - ETA updates
   - Location tracking

4. **Earnings**
   - Real-time earnings tracking
   - Payment history
   - Payout requests
   - Tax documents

5. **Performance**
   - Ratings & reviews
   - Success rate
   - Response time
   - Completion time

### Key Differences from Customer App

- **Job Discovery**: Active searching for jobs
- **Status Updates**: Technician controls status
- **Navigation**: Built-in GPS navigation
- **Earnings**: Detailed financial tracking
- **Performance**: Stats and ratings dashboard
- **Verification**: Document and identity verification

### Business Logic

- Commission: 10% platform fee
- Payout: Weekly or on-demand
- Rating: Affects job priority
- Response Time: Tracked and displayed
- Cancellation: Penalties for frequent cancellations
