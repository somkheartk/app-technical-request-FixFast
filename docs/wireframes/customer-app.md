## Notes

### Screen Flow Summary

```
Splash → Welcome → Login/Register
  ↓
Home
  ├→ Categories → Create Request → Confirm → Track
  ├→ My Requests → Request Detail → Chat/Payment
  ├→ Chat List → Chat Detail
  └→ Profile → Settings/Edit Profile

Request Flow:
Create → Pending → Assigned → In Progress → Completed → Payment → Review
```

### Responsive Considerations

1. **Orientation Support**
   - Portrait (primary)
   - Landscape (limited screens)

2. **Device Sizes**
   - Small phones: 320-375px
   - Standard phones: 375-414px
   - Large phones: 414-480px
   - Tablets: 768px+

3. **Safe Areas**
   - Top notch/status bar
   - Bottom home indicator
   - Side bezels

### Accessibility Features

- Large touch targets (min 48x48px)
- High contrast text
- Screen reader support
- Voice commands (future)
- Haptic feedback
